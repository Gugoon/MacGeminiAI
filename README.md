# 주요 기능 및 특징
<span style="color:orange">
사용자가 자연어로 질문하고 명령을 내릴 수 있는 채팅형 AI 앱

1. Mac OS 상단 아이콘 클릭하여 실행 가능
2. Image File을 드래그 드랍하여 업로드
3. 채팅 Message MarkDown 형식 지원
4. 채팅 히스토리 저장 및 열람

</span>
